\# 🧑‍💼 Employee Management System



A scalable \*\*Spring Boot-based Employee Management System\*\* with secure authentication, role-based access control, and modular backend architecture following industry best practices.



\---



\## 🚀 Features



\- 👤 User Registration \& Login system  

\- 🔐 JWT-based authentication \& authorization  

\- 🛡️ Role-based access control (ADMIN / USER)  

\- 🧑‍💼 Employee CRUD operations (Create, Read, Update, Delete)  

\- 🏢 Department management module  

\- 🔍 Employee search functionality  

\- 🌐 RESTful API design  

\- 🖥️ Thymeleaf-based UI integration  

\- ⚠️ Global exception handling mechanism  

\- 📦 Layered architecture (Controller → Service → Repository)  



\---



\## 🛠️ Tech Stack



\- \*\*Backend:\*\* Java, Spring Boot  

\- \*\*Security:\*\* Spring Security, JWT  

\- \*\*Database:\*\* MySQL  

\- \*\*ORM:\*\* Spring Data JPA, Hibernate  

\- \*\*Frontend:\*\* Thymeleaf, HTML, CSS  

\- \*\*Build Tool:\*\* Maven
